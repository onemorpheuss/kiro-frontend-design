# Дизайн интерфейса

Мощь Kiro, позволяющая создавать отличительные интерфейсы производственного уровня и избегать
типичной эстетики "AI-помоев". Адаптировано для Kiro на основе опыта Anthropic в области `frontend-дизайна`.

## Что это делает
Kiro автоматически использует эти возможности при работе с интерфейсом. Он создает готовый
к использованию код с:

- Смелой, продуманной эстетической направленностью
- Отличительной типографикой и гармоничными цветовыми решениями
- Эффектной анимацией и тщательной визуальной детализацией
- Контекстно-зависимой, доступной и отзывчивой реализацией

## Использование

Просто опишите, что вы хотите создать:

```
"Создайте панель мониторинга для приложения для потоковой передачи музыки"
"Создайте целевую страницу для запуска AI security"
"Создайте панель настроек с темным режимом"
```

Киро выбирает четкое эстетическое направление и внедряет производственный код с вниманием к
деталям.

## Структура

```
фронтенд-разработка/
├── Силы.МД # сила проявится + обзор
├── README-файле.МД 
└ ─ управление─ /
    ├── 01-дизайн-мышление.МД # место первого эстетическое направление 
    ├── 02-эстетика-основные положения.МД # ядром эстетики ссылка
    ├── 03-типографика и цвет.МД # шрифты + цвет системы
    ├── 04-motion-and-animation.md # Управление движением + загрузкой страницы
    ├── 05-implementation.md # HTML/React/Vue/Tailwind, адаптивный, a11y
    └── 06-avoiding-ai-slop.md # Аудит + исправление общих ошибок ИИ
```

## Установить

Поместите эту папку в свой каталог Kiro powers и откройте панель Powers в среде IDE:

- На уровне пользователя: "~/.kiro/powers/installed/frontend-design/"
- Или опубликуйте папку в репозитории git и установите ее через панель Powers.
